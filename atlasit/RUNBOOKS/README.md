# AtlasIT Runbooks

Operational procedures and incident response guides.

## Available Runbooks

- [Incident Response](incident-response.md) - Security incident procedures
- [Deployment](deployment.md) - Production deployment checklist
- [Rollback](rollback.md) - Service rollback procedures
- [Monitoring](monitoring.md) - Alert response and troubleshooting

## Runbook Template

Each runbook should include:

1. **Purpose**: What this runbook covers
2. **Prerequisites**: Required access, tools, knowledge
3. **Steps**: Numbered, actionable steps
4. **Validation**: How to verify success
5. **Rollback**: Recovery procedures if needed
6. **Escalation**: When and how to escalate

## Contributing

Update runbooks when:

- New procedures are established
- Incidents reveal gaps in documentation
- System architecture changes
- Tools or platforms change
