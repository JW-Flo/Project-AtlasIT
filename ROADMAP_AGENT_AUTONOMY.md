# Roadmap: Agent Autonomy

## Phase 1: Foundation (Current)
- [x] Framework manifest and schema definitions
- [x] Evidence emission and aggregation
- [x] Drift detection
- [x] OPA policy enforcement
- [x] Agent routing worker
- [x] CI/CD integration (matrix, drift, OPA)
- [x] CodeQL security scanning

## Phase 2: Enhanced Routing (Q1 2026)
- [ ] Machine learning severity prediction
- [ ] Historical decision analysis
- [ ] Auto-merge for low-risk changes
- [ ] Feedback loop integration
- [ ] Advanced trigger patterns (regex, file content analysis)

## Phase 3: Multi-Tenant (Q2 2026)
- [ ] Tenant-specific routing rules
- [ ] Evidence segregation by tenant
- [ ] Custom policy enforcement per tenant
- [ ] Tenant compliance reporting

## Phase 4: Advanced Automation (Q3 2026)
- [ ] Automated remediation suggestions
- [ ] Self-healing drift detection
- [ ] Predictive compliance gaps
- [ ] Agent performance metrics
- [ ] A/B testing for routing strategies

## Phase 5: Full Autonomy (Q4 2026)
- [ ] Zero-human-intervention for routine changes
- [ ] Self-optimizing routing rules
- [ ] Continuous evidence verification
- [ ] Real-time compliance dashboards
- [ ] Autonomous security patching

## Success Metrics
- **Agent Efficiency:** % of PRs handled without human intervention
- **Evidence Coverage:** % of controls with valid evidence
- **Drift Frequency:** Days between drift detections
- **Policy Violations:** Count of OPA denies per month
- **MTTR:** Mean time to remediate drift or violations
