AtlasIT
AtlasIT is the essential IT backbone for fast-growing businesses without a dedicated IT team. Centralize user provisioning, SaaS account management, infrastructure access, reporting, and more—automate the core functions of an early-stage IT department with security and scalability in mind.

Key Features
	•	Automated User ProvisioningOnboard/offboard users across all connected SaaS and infrastructure platforms from one place.
	•	Centralized Account Access ManagementManage, audit, and revoke access to Google Workspace, Slack, AWS, and more without SSO/SCIM upsell requirements—using direct API integrations wherever possible.
	•	Integrated IdP (Identity Provider)Host an internal IdP (OIDC/SAML) if needed, or connect to your existing SSO.
	•	Comprehensive ReportingReal-time and historical reporting on user access, provisioning events, and compliance (GDPR, CCPA).
	•	Flexible IntegrationsModular support for HR, communications, infrastructure, security, device management, and more.
	•	Cloud-First, Local-ReadyBuilt for the cloud (multi-tenant, secure, low-cost), but can be deployed locally if required.
	•	Role-Based Access ControlGranular roles for admins, managers, auditors, and background-secured authentication.

Quick Start
Prerequisites
	•	Docker & Docker Compose (recommended)
	•	Node.js >= 18.x (if developing locally)
	•	Cloud provider account (Google Cloud, AWS, or local VM)
