# Review – Senior AI/DevOps Lens

* **First principles**: does the solution derive from fundamentals, not hacks?  
* **Complexity**: highlight any O(n²) hot paths; propose refactor.  
* **Cost**: note AI/model calls; check against \$50/mo budget.  
* **Security & Privacy**: confirm input validation, secret handling, UII retention.  
* **Observability**: ensure logs are structured JSON; metrics counters exist.  
* **Automation Drift**: flag any manual steps.  
* **Docs**: verify comments explain *why*, not *what*.
* **Testing**: check for property tests, invariants, and coverage.
