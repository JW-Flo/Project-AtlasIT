## Key Integrations Documented

### Core Integrations

1. **Slack** - Communication and notification automation
   - Webhook configuration
   - Message-to-issue conversion
   - Real-time event notifications

2. **GitHub/GitLab** - Version control and CI/CD integration
   - PR/MR workflow automation
   - Commit linking
   - Deployment tracking

3. **AI Agents** - Intelligent automation teammates
   - Onboarding automation
   - Task orchestration
   - Documentation generation
   - Compliance monitoring

4. **Okta** - Identity and access management
   - User provisioning/deprovisioning
   - Group synchronization
   - SSO integration
   - JML (Joiner/Mover/Leaver) automation

5. **Jira** - Project management integration
   - Bidirectional issue sync
   - Workflow mapping
   - Sprint tracking

### Integration Framework

- **Connector Toolkit** - Build custom adapters from OpenAPI schemas
- **Adapter Generator** - Scaffold new connectors automatically
- **Simulator** - Validate adapter contracts
- **Research Engine** - Normalize vendor documentation into schemas
- **Registry** - Central catalog of available connectors

### API Capabilities

- **Core Workers API** - Onboarding, orchestration, documentation
- **Connector API** - Standardized access to integrated tools
- **Future APIs** - Compliance, risk assessment, policy management (planned)

## Usage Examples

### Enable Slack Integration

```bash
export FEATURE_CONNECTOR_SLACK=1
wrangler secret put SLACK_WEBHOOK_URL
```

### Create Custom Adapter

```bash
npm run build:adapter-gen
node packages/adapter-gen/dist/cli.js gen \
  --schema contracts/examples/schema.json \
  --name "Tool Name" \
  --out adapters
```

### Deploy Workers

```bash
cd onboarding && wrangler deploy
cd ../ai-orchestrator && wrangler deploy
cd ../documentation-worker && wrangler deploy
```
